# **App Name**: Akshay V. — Portfolio

## Core Features:

- Static Content Display: Showcase personal information (name, bio, tagline, skills), experience entries, and detailed project descriptions as provided.
- Dynamic Navigation with Scroll Tracking: Implement left-panel navigation links (About, Experience, Projects) that visually highlight the active section based on the user's scroll position using IntersectionObserver.
- Social & Action Links: Prominently display social media icons (GitHub, LinkedIn, Email) and a dedicated button for resume access.
- Responsive Layout Adaptation: Ensure the two-column sticky layout transitions seamlessly to a single-column stacked layout on mobile devices (screens less than 860px wide).
- Interactive UI Animations: Incorporate smooth fade-up animations for content sections on page load with staggered delays, and a captivating radial cursor glow effect.
- AI-Powered Skill Matcher: A tool leveraging generative AI to analyze user-provided job descriptions and dynamically highlight the most relevant skills and project experiences from the portfolio.

## Style Guidelines:

- Background color: Dark navy (#0a192f) for a deep, modern aesthetic in a dark scheme.
- Primary color: Muted slate (#8892b0) for general text and informational elements, providing readability against the dark background.
- Accent color: Vibrant teal (#64ffda) to draw attention to interactive elements, active states, and key highlights.
- Body font: 'Calibre' for primary text content. Labels, tags, and monospaced elements: 'SF Mono'. Note: currently only Google Fonts are supported.
- Modern and minimalist line-art style icons for social media links (GitHub, LinkedIn, Email) and other navigational cues, ensuring consistency with the overall clean, tech-inspired design.
- A prominent two-column sticky layout on larger screens, featuring a fixed left navigation sidebar and a scrollable content area on the right. This layout transforms into a single-column stack on mobile devices for optimal viewing.
- Subtle, smooth fade-up entrance animations with staggered delays for various content sections on page load. Navigation items feature an animated line that expands and changes color to the accent teal on active or hover states, complemented by an interactive radial cursor glow effect.
